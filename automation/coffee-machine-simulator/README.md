# ☕ Coffee Machine Simulator (Python)

## 📌 Overview

A command-line simulation of a coffee machine that manages resources, processes payments, and serves drinks based on user input.

This project models a real-world system, handling inventory, transactions, and user interaction in a structured and modular way.

---

## 🚀 Features

* ☕ Supports multiple drinks (espresso, latte, cappuccino)
* 📊 Real-time resource tracking (water, milk, coffee, money)
* 💰 Coin-based payment system with change calculation
* ⚠️ Resource validation before processing orders
* 🧾 Report functionality for machine status

---

## 🧠 Key Concepts Demonstrated

* **System design and state management**
* **Working with structured data (dictionaries)**
* **Input validation and control flow**
* **Separation of concerns using functions**
* **Simulating real-world processes in code**
