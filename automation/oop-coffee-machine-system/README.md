# ☕ OOP Coffee Machine System (Python)

## 📌 Overview

An object-oriented simulation of a coffee machine built in Python.

This project models a real-world system using multiple interacting classes, handling menu selection, resource management, and payment processing in a structured and scalable way.

---

## 🚀 Features

* ☕ Supports multiple drinks (espresso, latte, cappuccino)
* 🧱 Fully object-oriented design with separate components
* 📊 Resource tracking (water, milk, coffee)
* 💰 Integrated payment system with coin processing and change handling
* 📋 Reporting functionality for machine status

---

## 🧠 Key Concepts Demonstrated

* **Object-Oriented Programming (OOP)**
* **Encapsulation and separation of concerns**
* **Class interaction and system design**
* **State management across multiple components**
* **Clean and modular code architecture**

## 🔍 Design Notes

The system is divided into independent components, each responsible for a specific part of the functionality:

* `Menu` handles available drinks
* `CoffeeMaker` manages machine resources
* `MoneyMachine` processes transactions

This separation improves maintainability, readability, and scalability.
