# 🎮 Hangman Game (Python)

## 📌 Overview

A command-line implementation of the classic **Hangman** game built in Python.
The program randomly selects a word and allows the user to guess letters while tracking remaining lives.

---

## 🚀 Features

* Random word selection from a predefined list
* Lives system with visual hangman stages (ASCII art)
* Tracks correct and incorrect guesses
* Reveals correctly guessed letters in their positions
* Ends when the player either guesses the word or runs out of lives

---

## 🧠 What This Project Demonstrates

* Use of loops and conditionals to control game flow
* Managing game state using lists (correct and incorrect guesses)
* Working with external Python files (`hangman_words`, `hangman_art`)
* Basic user input handling and validation

---

## 🛠 Tech Stack

* Python 3
