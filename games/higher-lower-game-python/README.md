# 🔝 Higher-Lower Game (Python)

## 📌 Overview

A command-line game where the player compares two entities and guesses which one has a higher follower count.

The game continues as long as the player answers correctly, increasing the score with each successful guess.

---

## 🚀 Features

* Random selection of entities from a dataset
* Continuous gameplay with score tracking
* Comparison-based decision making
* Dynamic progression (winning entry carries forward)
* Clear win/lose feedback

---

## 🧠 Key Concepts Demonstrated

* Working with structured data (lists of dictionaries)
* Maintaining and updating state across iterations
* Implementing game loops with progression
* Function abstraction for cleaner logic
* User input handling in CLI applications
